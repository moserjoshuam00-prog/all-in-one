# All-in-One Productivity App #

All-in-One Productivity App ist eine vielseitige App, die dir hilft, deinen Alltag effizient zu organisieren. Sie vereint Habit Tracker, Checkliste und Budgetplaner in einer einzigenAnwendung, damit du deine Ziele, Aufgaben und Finanzen zentral verwalten kannst.

Features

1. Habit Tracker
Verfolge deine täglichen Gewohnheiten.
Erhalte Erinnerungen und Motivation, um konsequent zu bleiben.
Visualisiere Fortschritte über Wochen und Monate.

3. Checkliste / To-Do-Liste
Erstelle Aufgabenlisten für Arbeit, Studium oder Alltag.
Setze Prioritäten und Fälligkeitstermine.
Markiere erledigte Aufgaben für ein befriedigendes Gefühl der Erfüllung.

5. Budget Planer
Verwalte Einnahmen und Ausgaben.
Kategorisiere Ausgaben für bessere Übersicht.
Erhalte wöchentliche oder monatliche Finanzübersichten.
